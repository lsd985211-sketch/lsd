# Changelog

All notable changes to this repository are documented here.

## 0.1.0

- Initial public seed system release.
- Added read-only bootstrap context generation.
- Added MCP-style wrapper for `bootstrap_context_pack`.
- Added origin, design, and example payload documentation.

## 0.2.0

- Added a canonical public architecture contract.
- Organized the seed system into iteration, resource, maintenance, and tool layers.
- Linked docs and skill framework pages to the shared contract.

## 0.3.0

- Added a machine-readable architecture schema and canonical contract module.
- Exposed the architecture contract through the public seed-system package and MCP wrapper.
- Expanded the bootstrap payload to include canonical artifacts and schema metadata.

## 0.4.0

- Added a machine-readable maintenance contract with snapshot, doctor, repair-plan, validate, and metrics.
- Exposed the maintenance contract as a read-only MCP tool.
- Bumped bootstrap and server versioning to keep the public contract aligned.

## 0.8.0

- Added the public capability index, agent capability list, growth model, and agent primitive groups.
- Exposed growth and primitive contracts through read-only MCP tools.
- Expanded bootstrap packs so generic agents can discover how to observe, orient, act, verify, and retain reusable lessons.

## 0.9.0

- Added reusable public tool and knowledge patterns.
- Exposed tool knowledge through the bootstrap pack and read-only MCP tools.
- Added guidance for code navigation, safe editing, verification, source checking, tool discovery, GUI/browser use, Git/release discipline, and memory/skill retention.

## 1.0.0

- Declared the public API surface.
- Added split contract schemas, guardrails, a tool catalog, and lightweight validation helpers.
- Exposed `public_api`, `tool_catalog`, `guardrails_contract`, and `contract_validate` read-only MCP tools.
